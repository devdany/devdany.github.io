Category: Categories: DataBase

mongodb를 설치하기!

요새 MEAN스택에 급 꽂혔습니다. 그래서 MongoDB를 설치해서 공부해보려고 합니다. 일단 설치를 해야겠죠!

전 Homebrew를 사용하기로 했습니다. 가장 간단하더라구요

터미널에서

brew install mongodb

설치 끝이네요.

대신, mongoDB에서 데이터를 저장할 디렉터리를 하나 생성해 줘야 합니다.

sudo mkdir -p /data/db

이제 디렉터리에 권한을 줘야겠죠

chown 사용자이름 /data/db

환경설정 끝입니다! 이제 키고 접속해볼까요?

mongod

이걸로 mongoDB가 활성화 됩니다. 이제 접속만 하면 되겠군요.

새로운 터미널을 하나 열어서

mongo

접속 끝입니다. 일단 시작이 간편해서 좋네요. 공부는 간편하지 않겠지만요…
